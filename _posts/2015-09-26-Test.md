Matlab 中典型的绘图过程，通常要经过6个步骤。
1.	数据准备。
2.	设置当前绘图区
3.	绘图。
4.	设置坐标轴和网格属性
5.	标注图形
6.	保存和导出图形

{% highlight matlab %}
clear all;
x = 0:0.01:3*pi;
y1 = sin(x);
y2 = sin(3*x);
y3 = sin(5*x);
{% endhighlight %}
